# personal_portfolio

Personal portfolio built with Flutter and GetX. The app presents a landing page with hero, projects, skills, contact, and footer sections, optimized for responsive layouts.

## Stack

- Flutter
- Dart
- GetX
- Google Fonts
- url_launcher

## Run Locally

```bash
flutter pub get
flutter run
```

## Quality Check

```bash
flutter analyze
```

## Project Structure

```text
lib/
  app/
    models/
    modules/home/
      bindings/
      controllers/
      views/
      widgets/
    routes/
    services/
```
